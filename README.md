# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/your-repository-name.git
cd your-repository-name
```

Install dependencies:

```bash
pip install -r requirements.txt
```

```bash
python app.py
```

```bash
http://127.0.0.1:7860
```
